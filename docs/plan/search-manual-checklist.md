# Search Manual Checklist (Cycle 25)

목표
- 한국어 검색 품질(`tsvector` + `pg_trgm`) 적용 후 대표 질의 정확도를 수동 검증한다.
- 최소 20개 대표 시나리오를 점검하고, 오탐/미탐을 기록한다.

테스트 환경
- URL: `/search`
- 데이터: 최근 2주 게시글 + 병원/장소/자유글 혼합
- 범위: `전체/제목/내용/작성자`
- 확인 기준: 상위 10개 결과에서 기대 문서가 몇 위에 노출되는지

체크 규칙
- `PASS`: 기대 결과가 상위 3위 이내
- `WARN`: 기대 결과가 4~10위
- `FAIL`: 상위 10위 밖 또는 오탐 과다

## 회귀 체크

- [ ] `searchIn=TITLE`일 때 내용 매치가 과도하게 상위로 올라오지 않는다.
- [ ] `searchIn=AUTHOR`일 때 제목/내용 매치만으로 결과가 나오지 않는다.
- [ ] 게스트 차단 카테고리는 검색 결과에서 제외된다.
- [ ] 로컬/글로벌 스코프 전환 시 결과가 정책대로 분리된다.
- [ ] 결과 0건일 때 Empty state가 정상 노출된다.
