## 운영 화면 빠른 지도

- `/admin/reports`
  - 신고 큐, auto-hide, 제재 적용 상태 확인
- `/admin/auth-audits`
  - 로그인/비밀번호/회원가입 감사 로그 확인
- `/admin/policies`
  - 운영 정책과 개인화 튜닝 정책 조정
- `/admin/personalization`
  - personalized feed CTR, 광고 반응, audience key 지표 확인
- `/admin/breeds`
  - 품종 사전 override/custom row 관리
- `/saved`
  - 사용자 저장 글 확인, 지원 응대 시 북마크 상태 검토
